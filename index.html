    The MoMEMta project¶

    MoMEMta is a C++ software package to compute Matrix Element weights. Designed in a modular way, it covers the needs of experimental analysis workflows at the LHC. MoMEMta provides working examples for some common final states (, WW, …). Expert user or not, be prepared to feel the freedom of configuring your MEM computation at all levels!

    Note

    If you're using MoMEMta for your research, please make sure you're citing our reference paper!

    MoMEMta is based on:

    • C++, ROOT, Lua scripting language
    • Cuba (Monte-Carlo integration library)
    • External PDFs (LHAPDF by default)
    • External Matrix Elements (currently provided by our MadGraph C++ exporter plugin)

    You will probably want to start with our getting started guide, or check out our set of working examples.
